# Online-Book-Recommendation-System-
An online book recommendation and shopping system developed using HTML, CSS, JavaScript, PHP, and MySQL.

### 🏠 Home Page
The Home Page provides a modern and user-friendly interface for users to explore books online. Users can search books by title or author name using the integrated search functionality with autocomplete suggestions. The responsive design ensures smooth navigation across different devices.

---

### 🔍 Search Functionality
The Search Module allows users to quickly find books using keywords, author names, or titles. Dynamic search suggestions improve user experience and help users discover books efficiently.

---

### 📖 Book Details Page
The Book Details Page displays complete information about selected books including:
- Book Title
- Author Name
- ISBN Number
- Ratings
- Price
- Description
- Book Cover Image

Users can also add books to their cart or favorites directly from this page.

---

### ❤️ Favorites / Wishlist
The Favorites Module enables users to save books for future reference. This feature helps users create a personalized reading collection and improves user engagement.

---

### 🛒 Cart Management
The Cart Management Module allows users to:
- Add books to cart
- Update quantity
- Remove books from cart
- View total cost
- Proceed to checkout

This module ensures a smooth and interactive shopping experience.

---

### 💳 Checkout & Payment Page
The Checkout Module collects user delivery details and provides multiple payment options such as:
- UPI
- Credit/Debit Card
- BHIM
- Cash on Delivery (COD)

The payment process is designed to be simple, secure, and user-friendly.

---

### 📦 Order Management
The Order Module helps users track their orders by displaying:
- Order ID
- Order Date
- Payment Method
- Total Cost

Users can view their complete order history within the application.

---

### 👤 User Account Module
The User Account Section includes:
- My Account
- My Orders
- Logout Option

It allows users to manage personal information and monitor previous purchases.

---

### 📱 Responsive Design
The complete application is developed using Bootstrap to ensure responsiveness and compatibility across desktops, tablets, and mobile devices.
